> Typora是一款非常实用的Markdown阅读器 
破解方式如下  
- 先去[Typora官网](https://typoraio.cn/)下载原软件
- 打开'/Applications/Typora.app/Contents/Resources/TypeMark/page-dist/static/js/' 下的Licenselndex.180dd4c7.---.chunk.js 文件
- 搜索 hasActivated="true"==e.hasActivated 将e.hasActivated替换为"true"
- 打开'/Applications/Typora.app/Contents/Resources/TypeMark/page-dist/static/js/'下的0.99879679.chunk.js文件
- 文件头部添加 setTimeout(function() { document.querySelector('.default-btn.secondary-btn').click(); }, 256);

 
 > 一些Typora的快捷方式  
 
| 源代码模式       | Command + /                 |
| ---------------- | --------------------------- |
| 切换侧边栏       | Command + Shift + L         |
| 超链接           | Command + K                 |
| 插入图像         | Command + Control + I       |
| 代码块           | Command + Control + C       |
| 引用             | Command + Option + Q        |
| 添加行（在表中） | Command  + Enter            |
| 删除行（在表中） | Command + Shift + Backspace |
